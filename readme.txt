Project Title: Database for Animal Doctors (Mar-Jul 2021)

Overview: This project involves designing and implementing a database which
can be used to support a variety of activities conducted by a Veterinary Doctor.

My Role:
- Designing the database using a model.
- Implementing the model into a mySQL database.
- Use GitLab with regular commits to show a steady progression.

Outcomes:
- Achieved a satisfactory conceptual model based on the task. (ERD
diagram)
- Created a UML model with respect to the ERD diagram.
- Successfully implemented the design of the UML model into mySQL
database.
